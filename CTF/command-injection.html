Actions allowed panel

Moving files into tmp

Moving a file into /var/www/html/files/tmp succeeds without error.

Blocked path outside tmp

Moving a file behind /tmp triggers a malicious block error.

Intercepting and analyzing the request

The move command is passed in the URL after index.php. We can inject additional commands using operators such as &.

Operator reference

Operator Character URL-encoded Behavior
Semicolon;%3bboth commands
New line\n%0aboth commands
Background&%26both; second output first
Pipe|%7conly second output
AND&&%26%26both; second if first succeeds
OR||%7c%7csecond if first fails
Sub-shell backticks``%60%60both (Linux)
Sub-shell $()$()%24%28%29both (Linux)

Bypass with obfuscation

The whoami command is blocked so we escape and use quotes and variable expansion to run it as www-data.
